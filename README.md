## Proyecto: Sistema para el Control de Servicios Educativos Comunitarios

## Descripción

Este sistema está diseñado para el control de Servicios Educativos Comunitarios en el estado de Zacatecas. Utiliza información del INEGI para gestionar una base de datos de localidades y servicios educativos en los niveles de Preescolar, Primaria y Secundaria.

### **Base de Datos**

El sistema contempla inicialmente las siguientes tablas:
- Localidades INEGI
- Servicios educativos
- Catálogo de programas de educación básica

Cada servicio educativo se identifica mediante una llave compuesta por el identificador de localidad del INEGI y la clave del programa educativo.

### **Funcionalidades**

- Registro de altas y clausuras de servicios educativos por ciclo escolar.
- Gestión del estatus de los servicios: activo, baja, clausura o reapertura.

### **Alcance**

El sistema permite mantener un control actualizado y detallado de los servicios educativos comunitarios, facilitando la administración y seguimiento de los mismos en la entidad de Zacatecas.
